# mini-time-tracker
A stylish and simple time tracking application developed as a test task for Viso Academy. The application allows users to log work hours, view history, and manage daily tasks with a cozy "Espresso Shot" design theme.
### Features
• Time Logging: Add hours, dates, and descriptions for specific projects.

• History View: Browse entries by date with daily totals.

• Grand Total: View cumulative hours for all time.

• CRUD Operations: Create, Read, Update, and Delete time entries.

• Custom Design: Unique "Espresso Shot" theme using CSS variables and BEM methodology.

### How to run
The project is divided into two parts: frontend and backend, you need to write *npm install* in each folder and then *npm run dev*
